# Rule-Based Chatbot using Python

##  Project Description
This project is a simple rule-based chatbot developed using Python.  
The chatbot responds to user inputs based on predefined rules and keywords.

##  Features
- Responds to greetings like "hello" and "hi"
- Handles exit commands such as "exit", "bye"
- Provides learning-related responses
- Simple and beginner-friendly Python logic

##  Technologies Used
- Python 3
